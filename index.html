Web Development: developed a "Guess the Next Letter" game

I created a fun and challenging game called "Guess the Next Letter." The game is made using HTML, CSS, and JavaScript, and it has two modes: choice and typing. The player is also limited by a timer, adding an extra level of excitement to the game. Additionally, if the player gets a wrong answer, the game will provide the correct answer, and if the timer runs out, the game will give the player their score. The game is optimized for desktop view only and has a CSS usage of 43%, JavaScript usage of 30.6%, and HTML usage of 26.4%, as per the stats from Github. It was a fulfilling experience to create and play the game, even though it is not responsive. I'm looking forward to remaking it using Bootstrap to improve its functionality and accessibility.

Technologies used: HTML5, CSS3, JavaScript, Github, Git, Google Fonts, Facebook Chat Plugin
